============
Installation
============

At the command line::

    $ tar xzf vim-manager.X.Y.Z.tgz
    $ cd vim_manager.X.Y.Z
    $ pip install .

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv vim-manager -p python3.7
    $ tar xzf vim-manager.X.Y.Z.tgz
    $ cd vim_manager.X.Y.Z
    $ pip install .
